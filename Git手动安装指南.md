# Git手动安装指南

## 步骤1：下载Git安装包

请按照以下步骤下载Git安装包：

1. 打开浏览器
2. 访问 [Git官方网站](https://git-scm.com/download/win)
3. 网站会自动检测您的操作系统并提供合适的下载链接
4. 点击下载按钮，等待下载完成

## 步骤2：运行安装程序

1. 打开您的「下载」文件夹
2. 找到名为 `Git-2.xx.x-64-bit.exe`（版本号可能不同）的安装文件
3. 双击运行这个文件

## 步骤3：按照安装向导进行配置

请按照以下截图指南配置Git安装：

### 步骤3.1：选择安装路径
- 保持默认路径：`C:\Program Files\Git`
- 点击「Next」继续

### 步骤3.2：选择安装组件
- 保持所有默认组件选中
- 确保「Git GUI Here」和「Git Bash Here」选项已选中
- 点击「Next」继续

### 步骤3.3：选择开始菜单文件夹
- 保持默认值
- 点击「Next」继续

### 步骤3.4：选择默认编辑器
- 建议选择：Visual Studio Code（如果已安装）
- 或者保持默认的Vim编辑器
- 点击「Next」继续

### 步骤3.5：调整PATH环境变量
- 选择："Git from the command line and also from 3rd-party software"
- 这个选项允许您在任何命令行工具中使用Git
- 点击「Next」继续

### 步骤3.6：选择HTTPS传输后端
- 选择："Use the OpenSSL library"
- 点击「Next」继续

### 步骤3.7：配置行尾符号处理
- 选择："Checkout Windows-style, commit Unix-style line endings"
- 点击「Next」继续

### 步骤3.8：选择终端模拟器
- 选择："Use Windows' default console window"
- 点击「Next」继续

### 步骤3.9：配置额外选项
- 保持默认选项
- 点击「Install」开始安装

## 步骤4：完成安装

1. 安装过程完成后，勾选「Launch Git Bash」选项
2. 点击「Finish」完成安装

## 步骤5：验证安装是否成功

Git Bash窗口会自动打开，输入以下命令验证Git是否正确安装：

```bash
git --version
```

如果安装成功，会显示类似以下的输出：
```
git version 2.43.0.windows.1
```

## 步骤6：重新启动计算机

为了确保Git完全集成到系统环境中，请重启您的计算机。

## 安装完成后

安装完成并重启计算机后，您可以按照之前创建的《GitHub完整推送指南.md》进行代码推送操作。

如果您在安装过程中遇到任何问题，请参考以下常见问题解决方案：

## 常见问题解决

1. **找不到Git命令**：如果在命令提示符中输入`git`显示「不是内部或外部命令」，请检查是否正确安装，以及系统PATH是否包含Git的安装路径。

2. **权限问题**：安装过程中如果出现权限错误，请尝试右键点击安装文件并选择「以管理员身份运行」。

3. **网络问题**：如果下载失败，请检查网络连接，或尝试使用VPN。

4. **安装后命令不生效**：确保安装完成后重新启动了计算机，这对于环境变量的正确配置非常重要。

祝您安装顺利！如有其他问题，请查阅官方文档或咨询技术支持。
